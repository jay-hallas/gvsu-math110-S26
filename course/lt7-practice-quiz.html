    Worksheet Learning Target 6: Systems of Linear Equations
    

    Objectives

    • Given a system of linear equations, or a word situation, or data sets create a system of equations as applicable and solve the system.
      🔗
    🔗

    1. Hitting a Protein Goal.

    One scoop of protein powder provides \(24\) grams of protein and \(120\) calories. One granola bar provides \(10\) grams of protein and \(200\) calories. You want a snack made of scoops and bars that provides exactly \(68\) grams of protein and exactly \(640\) calories.
    🔗

    (a)

    Define your variables and write a system of equations that models this situation.
    🔗
    Answer.
    Let \(p\) be the number of scoops of protein powder and \(b\) be the number of granola bars.
    🔗
    The system is:
    🔗
    \begin{gather*} 24p + 10b = 68\\ 120p + 200b = 640 \end{gather*}
    🔗
    The first equation counts grams of protein, and the second counts calories.
    🔗
    🔗
    🔗

    (b)

    Graph both equations in Desmos and identify the solution.
    🔗
    Answer.
    In Desmos, enter the equations using \(x\) for scoops and \(y\) for bars. The two lines cross at \((2, 2)\text{,}\) so the solution is \(p = 2\) and \(b = 2\text{.}\)
    🔗
    🔗
    🔗

    (c)

    Solve the system algebraically to confirm your answer.
    🔗
    Answer.
    Multiply the first equation by \(5\) so the protein-powder terms match the second equation:
    🔗
    \(120p + 50b = 340\)
    🔗
    Subtract this from the calorie equation:
    🔗
    \((120p + 200b) - (120p + 50b) = 640 - 340\)
    🔗
    \(150b = 300\text{,}\) so \(b = 2\text{.}\)
    🔗
    Substitute \(b = 2\) into the protein equation: \(24p + 10(2) = 68\text{,}\) so \(24p = 48\) and \(p = 2\text{.}\)
    🔗
    🔗
    🔗

    (d)

    Interpret the solution in the context of the situation.
    🔗
    Answer.
    You should use \(2\) scoops of protein powder and \(2\) granola bars to get exactly \(68\) grams of protein and \(640\) calories.
    🔗
    🔗
    🔗
    🔗

    2. Twos and Threes.

    In a game, a basketball player makes \(12\) shots from the field, each one a two-point shot or a three-point shot, and scores \(27\) points in total. You want to find how many of each kind of shot the player made.
    🔗

    (a)

    Define your variables and write a system of equations that models this situation.
    🔗
    Answer.
    Let \(x\) be the number of two-point shots and \(y\) be the number of three-point shots.
    🔗
    The system is:
    🔗
    \begin{gather*} x + y = 12\\ 2x + 3y = 27 \end{gather*}
    🔗
    The first equation counts shots, and the second counts points.
    🔗
    🔗
    🔗

    (b)

    Graph both equations in Desmos and identify the solution.
    🔗
    Answer.
    Graphing both lines in Desmos, they cross at \((9, 3)\text{,}\) so the solution is \(x = 9\) and \(y = 3\text{.}\)
    🔗
    🔗
    🔗

    (c)

    Solve the system algebraically to confirm your answer.
    🔗
    Answer.
    Solve the first equation for \(x\text{:}\) \(x = 12 - y\text{.}\)
    🔗
    Substitute into the points equation:
    🔗
    \(2(12 - y) + 3y = 27\)
    🔗
    \(24 - 2y + 3y = 27\)
    🔗
    \(24 + y = 27\text{,}\) so \(y = 3\text{.}\)
    🔗
    Then \(x = 12 - 3 = 9\text{.}\)
    🔗
    🔗
    🔗

    (d)

    Interpret the solution in the context of the situation.
    🔗
    Answer.
    The player made \(9\) two-point shots and \(3\) three-point shots.
    🔗
    🔗
    🔗
    🔗

    3. Selling Show Tickets.

    A concert sells two kinds of tickets: general admission for \(\$40\) each and VIP for \(\$90\) each. The show sells \(200\) tickets in all and brings in \(\$11{,}000\text{.}\) You want to find how many of each kind of ticket were sold.
    🔗

    (a)

    Define your variables and write a system of equations that models this situation.
    🔗
    Answer.
    Let \(g\) be the number of general-admission tickets and \(v\) be the number of VIP tickets.
    🔗
    The system is:
    🔗
    \begin{gather*} g + v = 200\\ 40g + 90v = 11000 \end{gather*}
    🔗
    The first equation counts tickets, and the second counts dollars.
    🔗
    🔗
    🔗

    (b)

    Graph both equations in Desmos and identify the solution.
    🔗
    Answer.
    Graphing both lines in Desmos, they cross at \((140, 60)\text{,}\) so the solution is \(g = 140\) and \(v = 60\text{.}\)
    🔗
    🔗
    🔗

    (c)

    Solve the system algebraically to confirm your answer.
    🔗
    Answer.
    Solve the first equation for \(g\text{:}\) \(g = 200 - v\text{.}\)
    🔗
    Substitute into the dollars equation:
    🔗
    \(40(200 - v) + 90v = 11000\)
    🔗
    \(8000 - 40v + 90v = 11000\)
    🔗
    \(8000 + 50v = 11000\)
    🔗
    \(50v = 3000\text{,}\) so \(v = 60\text{.}\)
    🔗
    Then \(g = 200 - 60 = 140\text{.}\)
    🔗
    🔗
    🔗

    (d)

    Interpret the solution in the context of the situation.
    🔗
    Answer.
    The concert sold \(140\) general-admission tickets and \(60\) VIP tickets.
    🔗
    🔗
    🔗
    🔗

    4. How Many Solutions?

    Consider the following system of linear equations:
    🔗
    \begin{gather*} 2x + 3y = 6\\ 4x + 6y = 18 \end{gather*}
    🔗

    (a)

    Graph both equations in Desmos. How many solutions does the system have? Explain what the graph shows.
    🔗
    Answer.
    Rewriting each equation in slope-intercept form gives \(y = -\frac{2}{3}x + 2\) and \(y = -\frac{2}{3}x + 3\text{.}\) Both lines have slope \(-\frac{2}{3}\) but different vertical intercepts (\(2\) and \(3\)), so they are parallel and never cross. The system has no solution.
    🔗
    🔗
    🔗

    (b)

    Solve the system algebraically. How does the algebra confirm your answer from the previous part?
    🔗
    Answer.
    Multiply the first equation by \(2\text{:}\) \(4x + 6y = 12\text{.}\) Subtract this from the second equation:
    🔗
    \((4x + 6y) - (4x + 6y) = 18 - 12\)
    🔗
    \(0 = 6\)
    🔗
    This statement is false, which means no pair \((x, y)\) can satisfy both equations. That agrees with the graph: the lines are parallel, so there is no solution.
    🔗
    🔗
    🔗

    (c)

    Change the constant on the right side of the second equation from \(18\) to a new value \(A\) so that the system has infinitely many solutions. What is \(A\text{,}\) and how do you know?
    🔗
    Answer.
    Notice that the left side of the second equation is twice the left side of the first, since \(4x + 6y = 2(2x + 3y)\text{.}\) For the two equations to describe the same line, the right side must follow the same pattern: \(A = 2(6) = 12\text{.}\)
    🔗
    With \(A = 12\text{,}\) the second equation is exactly \(2\) times the first, so both equations represent the same line. Every point on that line satisfies both equations, so the system has infinitely many solutions.
    🔗
    🔗
    🔗

    (d)

    Is there any value of \(A\) that makes the system have exactly one solution? Explain.
    🔗
    Answer.
    No. The left side of the second equation is always twice the left side of the first, so the two lines always have the same slope. Changing \(A\) only shifts the second line up or down. The lines are therefore either parallel and separate (no solution, when \(A \neq 12\)) or the very same line (infinitely many solutions, when \(A = 12\)). Lines with equal slopes never cross at a single point, so no value of \(A\) can give exactly one solution.
    🔗
